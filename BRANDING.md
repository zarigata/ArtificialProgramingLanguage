- **Electric Blue**: `#0080FF` - Main brand color

- **Deep Purple**: `#6B2FFF` - Secondary, AI theme

- **Dark Gray**: `#1A1A1A` - Backgrounds
- **Light Gray**: `#E0E0E0` - Text on dark
- **White**: `#FFFFFF` - Clean backgrounds
- **Black**: `#000000` - Text, contrast

#### Gradient Options
- Blue to Cyan (speed, flow)
- Purple to Blue (AI, technology)
- Dark to Light (depth, clarity)

### Typography

#### Primary Font (Branding)
- **Sans-serif**, modern, geometric
- Bold weight for "VeZ"
- Examples: Inter, Poppins, Montserrat, Space Grotesk

#### Code Font (Documentation)
- **Monospace**, clear, readable
- Examples: JetBrains Mono, Fira Code, Cascadia Code

#### Recommended Pairings
- **Headings**: Space Grotesk Bold
- **Body**: Inter Regular
- **Code**: JetBrains Mono

## Brand Voice

### Tone
- **Confident**: We're building something revolutionary

- **Accessible**: Complex ideas, clear communication
- **Forward-thinking**: Future-focused, innovative

### Writing Style
- Direct and concise

- Inclusive language
- Active voice preferred
